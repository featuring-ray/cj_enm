# CJ × 피처링 온스타일 크리에이터 플랫폼

## 프로젝트 개요

CJ 온스타일의 크리에이터 커머스 생태계를 위한 3개 플랫폼 목업 구현 프로젝트.
모든 데이터는 Mock 데이터를 사용하며, 실제 API 연동 없이 프론트엔드 목업으로 구현한다.

### 플랫폼 구성

| 플랫폼 | 대상 | 설명 |
|--------|------|------|
| **온트너(Ontner)** | 크리에이터(인플루언서) | 크리에이터가 캠페인을 탐색하고 성과를 확인하는 서비스 |
| **온트러스트(Ontrust)** | CJ 내부 MD | MD가 크리에이터를 탐색·추천·관리하고 캠페인 성과를 분석하는 내부 도구 |
| **파트너(Partner)** | 외부 협력사/브랜드 | 협력사가 온트너 등록 크리에이터만 제한적으로 조회하는 서비스 |

### 핵심 용어

- **크리에이터**: 인스타그램/유튜브 인플루언서 (CJ 온스타일과 협업)
- **캠페인**: CJ가 운영하는 마케팅/판매 캠페인 (공동구매, 브랜드 협업 등)
- **어필리에이트/리워드 링크**: 크리에이터가 공유하는 성과 기반 수익 링크
- **공구 콘텐츠**: 공동구매 관련 인스타그램/유튜브 콘텐츠
- **크루찾기**: 팔로워 유사도 분석 기능 (베타)
- **인게이지먼트**: 좋아요, 댓글, 저장, 공유 등 콘텐츠 반응 지표
- **온트너 회원**: 온트너 플랫폼에 가입한 크리에이터 (전체 인플루언서의 일부)

---

## 기술 스택

- **프레임워크**: Next.js (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **컴포넌트**: shadcn/ui
- **차트**: Recharts
- **상태관리**: React Context (또는 Zustand)
- **Mock 데이터**: JSON 파일 또는 인메모리
- **아이콘**: Lucide React

## 화면 목록 및 상세 스펙

### 온트너 (Ontner) — 크리에이터용 플랫폼

#### ONT-001: 홈 대시보드 `/ontner/dashboard`
> **참고**: CJ 내부 자체 개발 범위이나, 목업에서는 레이아웃만 구현

- 캠페인 진행 현황 카드 (진행/신청/제안/완료 수)
- 캠페인 실적 그래프 (Recharts 라인 차트)
- 진행 중 캠페인별 콘텐츠 인게이지먼트 (평균)
- 최근 알림/공지사항 리스트

#### ONT-002: 프로필 관리 `/ontner/mypage`
- 크리에이터 프로필 카드
  - 프로필 이미지, 희망 카테고리, 희망 단가, RS, 자기소개
- SNS 계정 연동 정보 (인스타그램 핸들, 유튜브 핸들)
- 주요 캠페인 이력 (공개 설정된 캠페인만)
- "프로필 수정" 버튼 → ONT-003 모달
- "캠페인 공개 설정" 버튼 → ONT-004 모달

#### ONT-003: 프로필 수정 (모달)
- 프로필 이미지 업로드
- 희망 카테고리 선택 (드롭다운/태그)
- 희망 단가 입력
- RS 입력
- 자기소개 텍스트에리어

#### ONT-004: 캠페인 공개 설정 (모달)
- 참여한 캠페인 목록 체크리스트
- 선택/해제하여 프로필에 공개할 캠페인 결정

#### ONT-005: 캠페인 역제안 `/ontner/campaign/counter-proposal`
- 제안 목록 테이블: 제안일, 브랜드명, 수락/거절 여부, 날짜
- "제안하기" 버튼
  - 브랜드 선택 드롭다운 (추천 브랜드 상단 표시)
  - 제안 내용 입력 폼
- 브랜드 추천 영역: 내가 진행한 캠페인/어필리에이트 카테고리 기반 유사 브랜드 추천 리스트

#### ONT-006: 크루찾기 `/ontner/crew-finder`
> **핵심 기능 (베타)**

- 내 프로필 카드 (고정)
- 비교 대상 크리에이터 선택 (1명만, 풀 ~200명 내에서)
- "크리에이터 선택" 버튼 → ONT-007 모달
- 분석 결과 표시:
  - 팔로워 일치율 (%) — 큰 숫자로 강조
  - 비교 테이블: 나 vs 상대방 기본 정보
  - 참고: 온트너에서는 세그먼트 정보 미표시
- 분석 중 로딩 상태 UI (배치 처리 시뮬레이션)
- "결과 없음" 상태: 아직 분석되지 않은 경우

#### ONT-007: 크리에이터 선택 (모달)
- 크리에이터 검색/목록
- 팔로워 수집 완료된 크리에이터만 표시 (~200명)
- 선택 시 크루찾기 페이지에 추가

#### ONT-008: 캠페인 탐색/추천 `/ontner/campaign/explore`
- 탭 구분: "전체 캠페인" / "추천 캠페인"
- 캠페인 카드 리스트 (이미지, 브랜드, 기간, 상태, 보상)
- 추천 캠페인 영역:
  - 추천 사유 태그 표시 ("성과 유사 브랜드", "구매 고객 기반", "유사 크리에이터")
  - 배치 업데이트 기준 일시 표시
- 필터: 카테고리, 상태, 기간
- 캠페인 클릭 → ONT-009

#### ONT-009: 캠페인 상세 `/ontner/campaign/[id]`
- 캠페인 기본 정보 (브랜드, 기간, 보상, 상세 설명)
- 참여 신청 버튼
- "이 캠페인과 어울리는 콘텐츠" 추천 섹션
  - 인게이지먼트 상위 연관 콘텐츠 카드 리스트

#### ONT-010: 콘텐츠 탐색 `/ontner/content`
- "뜨는 공구 콘텐츠" Top 50 랭킹
  - 인게이지먼트 스코어 기준 정렬
  - 콘텐츠 썸네일, 크리에이터명, 좋아요/댓글/저장 수, 스코어
- 카테고리 필터 탭
- 콘텐츠 클릭 시: 연관 캠페인 목록 사이드 패널
- 캠페인 클릭 → ONT-009로 이동

#### ONT-011: 캠페인별 성과 조회 `/ontner/performance`
- 기간 선택 (DateRangePicker)
- 완료된 캠페인 드롭다운 선택
- 성과 요약 카드: 총 매출, 클릭수, 전환율, 콘텐츠 수
- 콘텐츠별 인게이지먼트 테이블
- 리워드 링크 성과 포함

#### ONT-012: 인사이트 리포트 (캠페인) `/ontner/insight/campaign`
- 기간 선택
- 캠페인 드롭다운 선택
- 통계 대시보드:
  - 콘텐츠 인게이지먼트 상위 Top 5 (바 차트)
  - 콘텐츠 유형별 성과 (파이 차트)
  - 시간대별 반응 추이 (라인 차트)
- CJ 데이터 영역 (Mock):
  - 성과 상위 캠페인 Top 5
  - 상위 상품, 재구매율
- 고정 텍스트 + 동적 숫자 (AI 텍스트 생성 없음)

#### ONT-013: 인사이트 리포트 (어필리에이트) `/ontner/insight/affiliate`
- ONT-012와 유사한 구조
- 어필리에이트(리워드 링크) 상품 기준 조회

### 온트러스트 (Ontrust) — MD용 내부 플랫폼

> 온트러스트 UI는 어드민 스타일 레이아웃 (사이드바 내비게이션 + 콘텐츠 영역)

#### OTR-001: 크리에이터 탐색 `/ontrust/creator/search`
> **최우선 기능**

- 검색바: 키워드 풀텍스트 검색
- 필터 패널:
  - 플랫폼: 인스타그램 / 유튜브 선택
  - 카테고리 선택
  - 팔로워 수 범위
  - ~~브랜드 필터~~ → 키워드 검색으로 대체 (필터 UI 없음, 검색어에 브랜드명 포함)
  - ~~공동구매 계정~~ → 키워드 검색으로 대체
  - ~~세일즈 단가 필터~~ → **불가** (데이터 없음)
  - ~~온트너 회원만 보기/제외~~ → **검색 필터 불가**, 결과 테이블에 "온트너 회원" 뱃지로만 표시
- 검색 결과 테이블:
  - 프로필 이미지, 핸들명, 팔로워 수, 인게이지먼트율
  - 온트너 회원 여부 뱃지
  - 세일즈 단가 컬럼 (데이터 있는 경우만)
  - 카테고리
- 행 액션: "상세 보기", "캠페인 제안", "북마크", "DM 발송"

#### OTR-002: 크리에이터 상세 리포트 `/ontrust/creator/[id]`
- 크리에이터 프로필 헤더 (온트너 정보 + 피처링 정보 통합)
- 오디언스 분석 섹션:
  - 성별/연령 분포 (바 차트)
  - 지역 분포
  - 참고 텍스트: "콘텐츠 반응 기준 예측 데이터 (실제 팔로워 기준과 다를 수 있음)"
- 콘텐츠 인게이지먼트 섹션:
  - 최근 콘텐츠 리스트 + 인게이지먼트 지표
  - 평균 인게이지먼트율 추이 (라인 차트)
- 캠페인 이력 (온트너 공개 캠페인)
- 하단 액션: "캠페인 제안", "북마크에 추가", "DM 발송"

#### OTR-003~004: 캠페인 제안 (모달 2단계)
- Step 1: 캠페인 목록에서 선택 (검색 + 리스트)
- Step 2: 제안 내용 입력 폼 → 제안 발신

#### OTR-005: 회원가입 제안 (모달)
- 비온트너 크리에이터에게 온트너 가입 유도
- 캠페인 선택 → DM/이메일로 가입 안내 발송

#### OTR-006: DM 발신 `/ontrust/dm`
- DM 발송 폼:
  - 수신자 선택 (크리에이터 검색)
  - 발송 계정 선택 드롭다운 (10개 이상)
  - 템플릿 선택 (사전 정의 템플릿)
  - 변수 치환 미리보기 ({크리에이터명}, {캠페인명}, {링크} 등)
  - 메시지 편집
  - 링크 첨부
- 발송 이력 테이블: 발송일시, 수신자, 템플릿, 상태(발송/읽음/실패)
- 템플릿 관리 탭: 템플릿 CRUD

#### OTR-007: 팔로워 유사도 분석 `/ontrust/creator/similarity`
- 크리에이터 2~5명 선택 (검색 + 태그)
- "분석 요청" 버튼 → 배치 처리 시뮬레이션 (로딩)
- 분석 결과:
  - 비교 매트릭스 테이블 (크리에이터 쌍별 일치율 %)
  - 세그먼트 정보 표시 (온트러스트에서는 표시됨)
  - 오디언스 분석 결과
  - 크리에이터 카테고리 비교
- 과거 분석 이력 조회

#### OTR-008: 콘텐츠 탐색 `/ontrust/creator/content`
- 뜨는 공구 콘텐츠 랭킹 (카테고리별)
- 오가닉/광고 구분 탭
- 콘텐츠 카드: 썸네일, 인게이지먼트 지표, 크리에이터명
- 콘텐츠 선택 시: 해당 크리에이터 정보 사이드 패널
- 크리에이터 상세로 이동 링크

#### OTR-009: 크리에이터 추천 `/ontrust/creator/recommend`
- 캠페인 선택 드롭다운
- 선택된 캠페인 기준 Top 10 크리에이터 추천 카드
- 추천 점수 표시 (인게이지먼트 + 매출 가중)
- "배치 업데이트: YYYY-MM-DD" 표시
- 각 카드에서 바로 "제안하기" 가능

#### OTR-010~014: 크리에이터 북마크 관리 `/ontrust/creator/bookmark`
- 그룹 사이드바 (최대 500명/그룹)
- 그룹 CRUD (추가/수정/삭제 모달)
- 그룹 내 크리에이터 목록 테이블
- 일괄 액션: 캠페인 제안, 회원가입 제안, DM 발송
- 그룹 클릭 시 → 커스텀 리포트 뷰 (그룹 내 크리에이터 종합 분석)

#### OTR-015: 캠페인 등록 내 추천 `/ontrust/campaign/new`
- 캠페인 등록 폼 (기본 필드)
- 카테고리 선택 시 → 하단에 "추천 크리에이터 Top 10" 자동 표시
- OTR-009와 연동

#### OTR-016~017: 성과 조회 `/ontrust/performance`
- 캠페인 종합 성과 조회:
  - 캠페인 선택 → 실적 집계 + 인게이지먼트 집계
  - 콘텐츠 트래킹 데이터 시각화 (시간대별 추이)

## 주의사항

- 모든 데이터는 Mock이다. API 호출 시뮬레이션은 `setTimeout`으로 딜레이만 넣는다.
- "불가" 표시된 필터(세일즈 단가, 온트너 회원 필터링)는 UI에 넣되, 비활성화 상태 + 툴팁으로 "데이터 미제공" 안내를 표시한다.
- 추천 알고리즘 결과는 Mock 데이터에서 미리 계산된 값으로 표시한다. 실제 ML 로직은 구현하지 않는다.
- 크루찾기(팔로워 유사도)의 "분석 중" 상태를 반드시 구현한다 (배치 처리 시뮬레이션).
- 인사이트 리포트에서 AI 텍스트 생성은 제외한다. 고정 텍스트 템플릿에 숫자만 동적으로 바인딩한다.
- 콘텐츠 트래킹 데이터는 시간대별 Mock 데이터를 생성하여 차트로 표시한다.
- 댓글 키워드 워드 클라우드는 Mock 키워드 빈도 데이터로 구현한다.
- 온트너/온트러스트/파트너는 같은 앱 내 라우팅으로 구분하되, 각각 독립적인 레이아웃을 가진다.
