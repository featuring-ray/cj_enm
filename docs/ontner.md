## 1. 회원관리

### O-A-01: 회원가입 `/ontner/signup`
> **유형**: Page | **R&R**: CJ 단독

**요구사항**: R001 | **상태**: 확정

**주요 기능:**
- 간편 가입 프로세스 (DM/메일 링크를 통한 가입 유도)
- 사업자등록증 업로드
- 계정정보 입력 (최소화된 인풋)
- 전자계약 동의
- 제3자 정보 제공 동의 (R009)
  - 가입 시 처리 / 기존 회원은 캠페인 참여 시 팝업 동의

**데이터 소스:**
- CJ: 회원 가입/관리 API (CJ 자체 개발)

> 참고: CJ 내부 자체 개발 범위. 목업에서는 가입 플로우 레이아웃만 구현

### O-B-01: 프로필 관리 `/ontner/mypage`
> **유형**: Page | **R&R**: 피처링+CJ 협업

**요구사항**: R045 | **상태**: R&R 협의

**주요 기능:**
- 크리에이터 프로필 카드
  - 프로필 이미지, 희망 카테고리, 희망 단가, RS, 자기소개
- SNS 계정 연동 정보 (인스타그램 핸들, 유튜브 핸들)
- 주요 캠페인 이력 (공개 설정된 캠페인만 표시)
  - 자신이 진행했던 온트너 캠페인 중 공개할 캠페인 선택 (기본값: 전체 비공개)
  - 공개 대상은 MD/파트너 동일 적용
- "프로필 수정" 버튼 → O-B-02 팝업
- "캠페인 공개 설정" 버튼 → O-B-03 팝업

**데이터 소스:**
- CJ: 프로필 조회/저장 API, 권한 설정/저장/조회 API
- 피처링: 기획/화면개발

**제약사항:**
- 온트너-피처링 크리에이터 매핑키 필요 (인스타: 아이디, 유튜브: @네임명)
- 사용자 입력 정보와 피처링 분석 데이터는 별도 저장

### O-B-02: 프로필 수정 (팝업)

## 2. 캠페인 탐색 관리

### O-C-03: 캠페인 탐색/추천 `/ontner/campaign/explore`
> **유형**: Page | **R&R**: 피처링 단독

**요구사항**: R033 | **상태**: 결정 필요

**주요 기능:**
1. **캠페인 목록 조회**
   - 상태값(모집중)인 캠페인 전체 호출
   - 브랜드명, 캠페인명, 대표 이미지, 보상 내용, 모집 마감일
   - **필터 상세:**
     - 카테고리: 복수 선택 가능 (뷰티, 패션, 식품, 리빙, 디지털 등), 기본값 "전체"
     - 리워드 타입: 단일 선택 (현금, 제품, 포인트, 복합), 기본값 "전체"
     - SNS 채널: 단일 선택 (인스타그램, 유튜브), 기본값 "전체"
     - 필터 초기화 버튼 제공
   - 정렬: 최신순 (기본), 마감임박순, 보상금액순

2. **캠페인 추천 목록** (O-C-06 모듈 임베드)
   - 탭 구분: "전체 캠페인" / "추천 캠페인"
   - 추천 사유 태그 표시 ("성과 유사 브랜드", "구매 고객 기반", "유사 크리에이터")
   - 배치 업데이트 기준 일시 표시

**데이터 소스:**
- CJ: 매출/구매 데이터
- 피처링: 인게이지먼트 데이터

### O-C-04: 캠페인 상세 `/ontner/campaign/[id]`
> **유형**: Page | **R&R**: 피처링+CJ 협업

**주요 기능:**
- 캠페인 기본 정보 (브랜드, 기간, 보상, 상세 설명)
- 참여 신청 버튼
- **연관 콘텐츠 추천 섹션** (캠페인 상세 내)
  - 인게이지먼트 상위 연관 콘텐츠 카드 리스트 (TOP 3~5)
  - **Fallback 로직** (순차적):
    1. 1순위: 해당 '상품'과 일치하는 내부 진행 이력 피드
    2. 2순위: 해당 '브랜드'와 연관된 내/외부 피드
    3. 3순위: 상품/브랜드가 없다면 '소분류 카테고리' 일치하는 인기 피드
    4. 3순위까지 0건이면 해당 UI 섹션 전체 숨김 (Display:none)
  - 콘텐츠 썸네일 클릭 시 원문 링크 (아웃링크) 제공

### O-C-05: 콘텐츠 탐색을 통한 캠페인 추천 `/ontner/campaign/content`
> **유형**: Page | **R&R**: 피처링+CJ 협업

**주요 기능:**
- 캠페인 탐색 메뉴 내 별도 탭 등으로 진입
- **뜨는 공구 콘텐츠 Top 50 랭킹**
  - 인게이지먼트 가중치 점수 기반 정렬 (댓글 5.0 > 공유 4.0 > 저장 3.0 > 조회수 2.0 > 좋아요 1.5 > 팔로워 0.01)
  - 카테고리별, 기간별(1주일, 1개월, 3개월, 6개월) 조회
  - 대댓글 자동화 툴(소셜비즈 등) 사용 여부 결과 리스트 내 표기
- **콘텐츠별 연관 캠페인 매핑**
  - 콘텐츠 1개당 1~2개 참여 가능 캠페인/리워드 링크 상품 매핑
  - 콘텐츠 카테고리/상품 키워드와 현재 모집 중인 캠페인 매칭
  - **매핑 0건 시**: "현재 참여 가능한 연관 캠페인이 없습니다" 텍스트 표시 (목록에서 제외하지 않음)
  - 바로 신청 버튼 제공 → 캠페인 상세(O-C-04) 또는 리워드 링크 신청 페이지로 이동
- **콘텐츠 컬럼 표기:**
  - 순위뱃지 (1~50), 원본 SNS썸네일, SNS플랫폼 아이콘, 주요 성과 지표 (댓글, 조회수, 좋아요)
  - 크리에이터 핵심 정보 (프로필, 핸들명, 팔로워 수)
  - 연관 캠페인/리워드 링크 카드 (1~2개)
- 캠페인 클릭 → O-C-04로 이동
- **조회 필터:**
  - 카테고리 포함/제외
  - 브랜드 일치/유사 브랜드군 포함 (키워드 검색으로 동작)
  - 공동구매 진행 콘텐츠만 보기 (키워드 검색 기반)
  - 기간별 필터 (1주/1개월/3개월/6개월)

**미결사항:**
- 콘텐츠 트래킹 비용: [옵션A] D+0 시간단위 + D+1~ 일단위 (120원/회) vs [옵션B] 전구간 일단위 (120원/일)

**데이터 소스:**
- 피처링: 콘텐츠 크롤링

### O-C-06: 캠페인 추천 (모듈)
> **유형**: Module | **R&R**: CJ+피처링 협업

**주요 기능:**
- **3가지 테마별 추천 캠페인 카드 노출** (쇼핑몰식 가로 롤링 UI)
  - 각 테마별 독립 캐러셀, 자동 롤링 없음 (사용자 스와이프/화살표로 탐색)
  - 테마당 최소 1개 ~ 최대 10개 캠페인 카드 노출
  - 추천 데이터가 0건인 테마는 해당 섹션 전체 숨김
  - "배치 업데이트: YYYY-MM-DD" 일시 표시, 24시간+ 경과 시 경고 배너

1. **과거 성과 기반 추천**
   - 크리에이터 본인이 과거 상위 성과(매출)를 냈던 캠페인과 유사한 카테고리/브랜드의 캠페인·상품 추천

2. **내 고객 리텐션 데이터 기반 추천**
   - 내 캠페인 구매 고객이 플랫폼에서 추가 구매한 상품 분석 → 연관 캠페인/어필리에이트 상품 추천

3. **유사 크리에이터 고객 기반 추천**
   - 나와 핏(규모, 카테고리)이 비슷한 크리에이터의 고객 구매 트렌드 기반 추천

**노출 영역:**
- 캠페인 탐색 메뉴 (O-C-03)
- 리워드 링크 상품 탐색 메뉴
- 캠페인 상세페이지 (O-C-04)
- 캠페인별 성과 조회 리포트 하단 (O-D-02)

**데이터 소스 (CJ 제공 필요):**
1. 캠페인 및 크리에이터별 주문 추적 데이터
2. 특정 회원의 추가 구매 이력 (리텐션 로그)
3. 고객 세그먼테이션 데이터 (성별, 나이)
4. 크리에이터 과거 성과(매출 체력) 데이터

### O-C-07: 브랜드 역제안 `/ontner/campaign/counter-proposal`

**R&R:**
- 피처링: '역제안 타깃 브랜드 리스트 API'만 개발
- CJ: 역제안 전용 화면 UI, 브랜드 검색, 사내 메신저 플로우 전체 자체 구축

   - 캠페인 기간 동안 콘텐츠 업로드 추이 + 인게이지먼트 추이 + 주문 수량 추이를 하나의 그래프로 통합 표시 (Recharts 라인 차트)

   - 콘텐츠 유형
   - 콘텐츠별 인게이지먼트 (좋아요, 댓글, 조회수 등)
   - 주요 반응 키워드
   - 시간/일자별 콘텐츠 업로드 현황

4. **매출 기준 상세 (CJ 데이터)**
   - 시간별 주문 수량 (전체, 옵션별) 및 매출 현황
   - 주문 고객 세그먼트 (연령, 성별)
   - 함께 구매한 상품 Top10
   - 취소/교환/반품 현황 (취교반율, 주요 사유)
   - **캠페인 종료 후 D+14 기준 최종 매출 확정** (취교반 반영)

**데이터 소스:**
- CJ: 진행중 캠페인 목록 조회 API, 캠페인 단건 상세 조회 API, 시간 단위 주문/매출 집계, 고객 세그먼트, 함께 구매한 상품, 취교반 현황
- 피처링: 콘텐츠 기본 정보, 시간 단위 콘텐츠 인게이지먼트, 콘텐츠 업로드 현황
